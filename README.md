How to run

1. Masuk ke aplikas (lewat menu log in)
2. Masukan daftar belanja (nama barang,jumlah, dan harga satuan)
3. List tersebut jika sudah selesai belanjanya bisa di centang agar terlihat apa yang sudah terbeli
4.jika daftar belanja sudah tidak terpakai lagi kita bisa mengeklik tombol hapus untuk menghapus daftar belanja yang sudah tidak terpakai.
