# 🛒 TemanBelanja

TemanBelanja adalah aplikasi berbasis **C++ dan WebUI** yang membantu pengguna dalam mengelola daftar belanja secara mudah dan interaktif.

---

## 🚀 How to Run

1. Clone atau download repository ini:

```bash
git clone https://github.com/FahmiZain16/TemanBelanja.git
cd TemanBelanja
```

2. Compile file C++:

```bash
g++ *.cpp -o app.exe -Iwebui-windows-gcc-x64/include -Lwebui-windows-gcc-x64 -lwebui-2
```

3. Jalankan aplikasi:

```bash
app.exe
```

## note : app pada app.exe bebas diganti.

## 📖 Manual Penggunaan

### 📝 1. Menambahkan Daftar Belanja

- Masukkan data barang:
  - Nama barang
  - Jumlah
  - Harga satuan

- Data akan masuk ke dalam daftar belanja

---

### ✅ 2. Menandai Barang

- Barang yang sudah dibeli dapat dicentang
- Ini membantu membedakan:
  - Barang yang sudah dibeli
  - Barang yang belum dibeli

---

### 🗑️ 3. Menghapus Daftar Belanja

- Jika daftar sudah tidak digunakan:
  - Klik tombol **hapus**

- Data akan dihapus dari daftar

---

## ⚠️ Catatan

- Pastikan semua file `.cpp` sudah di-compile
- Gunakan CMD / PowerShell untuk menjalankan aplikasi
- Pastikan file HTML berada di lokasi yang benar
