## 🚀 Quick Reference - How to Use

### Step 1: Compile
```bash
# Windows
build.bat

# Linux/macOS
chmod +x build.sh
./build.sh

# Manual
g++ -std=c++17 -O2 app.cpp -o app.exe -I./include
```

### Step 2: Run
```bash
# Windows
app.exe

# Linux/macOS
./app
```

### Step 3: Open Browser
- Automatic: Should open automatically at http://localhost:8080
- Manual: Open browser → http://localhost:8080

### Step 4: Login
