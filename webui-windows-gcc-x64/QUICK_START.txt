## 🚀 Quick Start - 3 Langkah Mudah

### Step 1: Compile
**Windows:**
```bash
# Double-click build.bat
# Atau ketik di command prompt:
build.bat
```

**Linux/macOS:**
```bash
# Terminal:
chmod +x build.sh
./build.sh
```

**Manual compile jika build script tidak bekerja:**
```bash
g++ -std=c++17 -O2 app.cpp -o app.exe -I./include -static-libgcc -static-libstdc++
```

### Step 2: Run
**Windows:**
```bash
app.exe
```

**Linux/macOS:**
```bash
./app
```

### Step 3: Login
- Browser otomatis membuka di `http://localhost:8080`
- Login dengan akun demo:
  - Email: `budi@example.com`
  - Password: `password123`

## 🔄 How It Works

### Arsitektur
```
┌──────────────────────────────────────┐
│    WebUI C++ Application (app.exe)   │
├──────────────────────────────────────┤
│  Frontend (HTML/JavaScript/CSS)      │
│  - Renders pages dan UI             │
│  - Handle user input                │
│  - CRUD operations lokal            │
├──────────────────────────────────────┤
│  Data Storage (Browser LocalStorage) │
│  - Persist semua data              │
│  - JSON format                      │
├──────────────────────────────────────┤
│  C++ Backend (Optional)              │
│  - Business logic                   │
│  - Data validation                  │
│  - WebUI event bindings             │
└──────────────────────────────────────┘
```

### Flow
1. User interact dengan HTML (form, button, link)
2. JavaScript handle event
3. CRUD operation ke localStorage
4. UI re-render dengan data baru
5. Optional: call C++ backend untuk validation

## 💡 Common Tasks

### Menambah Daftar Belanja Baru
1. Klik "Daftar Belanja" di dashboard
2. Klik "Buat Daftar Baru"
3. Input nama daftar
4. Klik "Buat"


2. Edit `app.js`:

## 🐛 Troubleshooting

### Problem: "g++ not found"
**Solution:**
- Windows: Install MinGW dari https://www.mingw-w64.org/
- Linux: `sudo apt-get install build-essential`
- macOS: `brew install gcc`

### Problem: Browser tidak membuka
**Solution:**
- Manual: Buka `http://localhost:8080` di browser
- Check port 8080 tidak terpakai: `netstat -ano | findstr :8080`
- Try port lain dengan edit app.cpp

### Problem: Data hilang setelah close
**Solution:**
- LocalStorage hanya simpan selama browser session
- Data akan hilang jika:
  - Clear cache/cookies
  - Private/Incognito mode
- Solution: Implement database backend

### Problem: CSS tidak load
**Solution:**
- Ensure `style.css` ada di folder yang sama dengan app.exe
- Hard refresh browser: Ctrl+Shift+R (Windows) atau Cmd+Shift+R (Mac)
- Check DevTools (F12 → Network) untuk CSS load status
